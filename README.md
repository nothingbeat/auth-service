# Auth Service

## Project Description
This is a complete Auth service implementation based on FastAPI framework, providing user authentication and authorization functions.

## Tech Stack
- **Backend Framework**: FastAPI
- **Database**: PostgreSQL
- **Cache**: Redis
- **Authentication**: JWT Token
- **Containerization**: Docker & Docker Compose

## Branch Structure
- \master\ - Main branch, containing project documentation
- \develop\ - Development branch, containing complete code
